This course introduces the students to basic web design and development concepts and skills that will allow them to build professional website and modify existing website using php and wordpress. The course strives to really make the student understand the core concepts behind the www language. This is an accelerated course for the students interested in learning techniques for creating professional websites using HTML & CSS.
